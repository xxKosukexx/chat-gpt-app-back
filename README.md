## Description

Echo(クリーンアーキテクチャ採用)でAPIを作成することで、GoでAPIを作成する関連の知見を得ることを目的にしたプロジェクトです。
内部では、Chat-GPT APIを使用しており、その辺りでいろいろできないか試行錯誤する予定のプロジェクトでもあります。
gRPCサーバーとなるようにしています。
